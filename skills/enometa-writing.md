# ENOMETA Writing Style v10

## 채널 정체성

**ENS·NOĒMA·META** = 존재·사유·초월
"존재와 사유, 그 경계를 초월하다"

**미션:** 지치고 힘든 사람들에게 닿는 글. 철학적 깊이와 동기부여가 공존.
**비주얼 미션 (v10 추가):** 모든 문장은 시각적 메타포로 변환 가능해야 한다. 글이 곧 영상의 설계도.

---

## 핵심 톤: "담백한 격조"

- 따뜻하게 닿되 과잉 없이
- 위로하되 설교하지 않음
- 감정 건드리되 물렁하지 않음
- 담백하되 가볍지 않게
- 논리적이면서 뜨겁게 각성

**한 문장 정의:**
"지친 사람 옆에 앉아서 같이 바다 보면서 말해주는 글. 단, 문장 하나하나에 무게가 있는."

---

## v10 핵심 변경사항 (v9 대비)

### 1. 데이터아트형 스타일 추가 (4번째 스타일)
- 숫자/데이터로 후킹 → 철학적 재정의 → 존재론적 각성
- 비주얼 시스템과 직결되는 글쓰기 방식
- 데이터가 감정이 되는 순간을 설계

### 2. 비주얼 매핑 인식 (Visual-Aware Writing)
- 모든 문장이 비주얼 어휘(Visual Vocabulary)로 변환 가능하도록 설계
- 추상적 개념을 쓸 때 시각적 메타포가 자연스럽게 떠오르는 문장
- "반복" → 궤도를 도는 이미지 / "각성" → 무언가가 벗어나는 이미지

### 3. 데이터 후킹 기법
- 질문이나 공감 대신 숫자/팩트로 시작
- 짧은 데이터 문장 2~3개로 사실을 제시
- 숫자 자체가 충격이 되어야 함

### 문장 밀도 원칙 (v9 유지)
- **어휘의 품격과 문장 밀도가 리듬보다 우선한다**
- 끊어치기만으로 깊이를 만들지 않는다. 한 문장의 무게로 깊이를 만든다.
- "근데" → "하지만" / "그런데"
- "느꼈다" → "원망했다", "외면했다" 등 구체적 동사
- 구어체보다 문학적 격조.

### 어휘 레벨 기준
- **허용:** 사유, 해독, 역학, 구조, 문법, 원초적, 탐구, 본질, 작동 원리, 루프, 자동화, 균열, 잠식
- **지향:** 일상에서 이해 가능하되 한 층 위의 어휘
- **금지:** 학술 논문체, 번역투, 과시적 한자어 남용

### 반복 어휘 경계
- 같은 접속사가 두 번 이상 나오면 구조를 재설계
- 같은 형용사/부사가 연속 문장에 나오면 교체
- 동일한 논지를 다른 표현으로 반복하는 것은 설명이지 깊이가 아니다

## 핵심 프레임워크: 골든서클

| 골든서클 | ENOMETA 구조 | 역할 |
|----------|--------------|------|
| **Why** | 후킹 + 철학적 재정의 | 왜 이게 중요한지, 왜 통념이 틀렸는지 |
| **How** | 논리 전개 + 펀치라인 | 어떻게 다르게 봐야 하는지 |
| **What** | 확장된 착지 | 결국 뭘 해야 하는지, 뭐가 진짜인지 |

---

## 구조 공식 (골든서클 기반 5단계)

```
[WHY]
1. 후킹 (질문/공감/데이터)
   → 바로 잡는다. 설명 없이.
   → 데이터아트형: "하루에 뇌가 처리하는 생각의 수. 6만 개."

2. 철학적 재정의
   → 통념을 뒤집는다.
   → 데이터아트형: "이건 습관이 아니다. 자동화다."

[HOW]
3. 논리 전개
   → 왜 그런지 구체적으로 설명

4. 펀치라인
   → 기억에 박히는 한 줄. 문장 밀도가 가장 높아야 하는 지점.

[WHAT]
5. 확장된 착지
   → 결국 뭘 해야 하는지. 본질로 마무리.
```

## 스타일 유형 (4가지)

### 1. 에세이형 (기본)
- **시점:** 관찰자 → "우리"에게 말하는 에세이
- **톤:** 담백한 분석 + 따뜻한 각성
- **구조:** 골든서클 5단계 그대로 적용

### 2. 독백형
- **시점:** 1인칭 내면 독백
- **톤:** 새벽에 창밖 보며 내뱉는 말. 절제된 감정. 문학적 격조.
- **구조:** 골든서클 유연 적용. 감정의 흐름이 구조를 이끈다.
- **특징:** 회상/성찰 어미, 자문자답, 여운으로 마무리

### 3. 대화형
- **시점:** "너"에게 직접 말을 거는 느낌
- **톤:** 따뜻한 직접성. 어깨 잡고 눈 맞추며 말하는 온도.
- **구조:** 골든서클 유연 적용. 대화의 리듬이 구조를 이끈다.
- **특징:** 반말, 질문 던지고 스스로 답하기, 마지막 한마디가 꽂힘

### 4. 데이터아트형 (v10 신규)
- **시점:** 관찰자. 데이터가 먼저 말하고, "우리"가 해석한다.
- **톤:** 차갑게 시작, 뜨겁게 끝난다. 데이터의 냉정함 → 존재론적 뜨거움.
- **느낌:** 다큐멘터리 나레이션 + Max Cooper 뮤직비디오
- **구조:** 데이터 제시 → 철학적 전환 → 존재론적 각성 (3단 전환)

#### 데이터아트형 3단 전환 구조

```
[1단계] 데이터 제시 — 차가운 사실
  짧은 문장. 숫자. 팩트. 감정 없이 던진다.
  → 비주얼: 숫자 카운팅, 차트 성장, 파티클 생성
  → 음악: 미니멀, sparse clicks

[2단계] 철학적 전환 — 데이터가 질문이 되는 순간
  "이건 ~이 아니다. ~이다." 재정의 문장.
  데이터가 존재론적 질문으로 변하는 변곡점.
  → 비주얼: 파티클 분리, 궤도 반복, 색상 변화
  → 음악: 텐션 빌드업

[3단계] 존재론적 각성 — 숫자 너머의 인간
  차가운 분석이 따뜻한 각성으로.
  마지막 문장은 숫자가 아닌 존재의 언어.
  → 비주얼: 궤도 이탈, 연쇄 점등, 공간 확장
  → 음악: 브레이크스루, 밝은 패드
```

#### 데이터아트형 문장 설계 원칙

```
1. 데이터 문장은 극도로 짧게
   "하루에 뇌가 처리하는 생각의 수."
   "6만 개."

2. 재정의 문장은 대비로
   "이건 습관이 아니다."
   "자동화다."

3. 비주얼이 보이는 문장
   "같은 불안. 같은 후회." → 파티클 궤도 반복
   "루프에 균열이 생긴다." → 시각적 파열 이미지

4. 전환점에 가장 강한 문장
   "생존 전략이 어느새 존재를 잠식한 것이다."

5. 마지막은 데이터를 넘어선 언어
   숫자(6만)로 시작 → "하나"로 끝남
```

### 스타일 선택 기준

| 상황 | 추천 스타일 |
|------|------------|
| 사회현상 분석, 통념 비판 | 에세이형 |
| 내면 성찰, 존재론적 질문, 회고 | 독백형 |
| 위로, 응원, 동기부여, 각성 | 대화형 |
| 숫자/데이터 주제, 과학+철학, 뇌과학, 사회 통계 | 데이터아트형 |
| 쇼츠 비주얼과 직결되는 콘텐츠 | 데이터아트형 |

## 비주얼 매핑 가이드 (v10 추가)

### 문장 → 비주얼 어휘 연결 패턴

```
숫자 제시          → counter_up, bar_chart_grow, particle_birth(count)
비율/퍼센트        → particle_split_ratio
반복/루프/패턴     → particle_orbit
전환/변화          → color_shift, flow_field 방향 변화
파괴/균열          → glitch_boundary
각성/깨달음        → particle_escape, light_source_birth
연쇄/확산          → particle_chain_awaken
질문/의문          → 형태 불안정, 글리치
자유/해방          → space_expand
소멸/잊힘          → particle_dissolve, darkness_creep
고독/고립          → particle_repel
연결/관계          → 파티클 간 연결선
```

### 비주얼 변곡점 설계

대본에는 최소 2~3개의 비주얼 변곡점(화면이 확 바뀌는 순간 = 감정의 전환점)이 있어야 한다.

## 허용/금지 표현

### 허용
- 감정에 닿는 문장, 공감 기반 질문, 논리적 각성
- 따뜻한 명령형, 본질적 단어, 격조 있는 자조
- 데이터 직접 제시 ("6만 개", "95퍼센트") (v10)
- 과학/기술 용어의 철학적 전용 ("루프", "자동화", "균열") (v10)
- 비주얼이 보이는 동사 ("잠식하다", "이탈하다", "수렴하다") (v10)

### 금지
- 싸구려 감정 호소, 과잉 수사, 클리셰
- 설교조, 뻔한 비유, 선배 꼰대 느낌
- 가짜로 깊은 척 — 유일한 금기
- 같은 접속사/어휘 반복, 애매한 형용사
- 데이터 출처 나열 ("연구에 따르면") (v10)
- 데이터 해석을 설명조로 풀기 ("이것은 ~를 의미합니다") (v10)

## 펀치라인 패턴

- **재정의형:** "그건 네 잠재력이 불편한 사람들이 하는 말이야."
- **결과 제시형:** "그때의 '괜찮아'가 네 인생을 늦춘 말이었다는 걸."
- **본질 정의형:** "네 안의 갈증과 결핍. 그게 네 인생의 진짜 방향이니까."
- **대비형:** "기계의 사유 구조는 그토록 탐구하려 했으면서 / 정작 내가 발 딛고 선 세상의 작동 원리에는 눈을 감았다."
- **자각형:** "해독조차 하지 않은 언어에 응답을 기대했던 것이다."
- **데이터 전환형 (v10):** "에너지를 아끼려는 생존 전략이 어느새 존재를 잠식한 것이다."
- **데이터 대비형 (v10):** "새로운 하루는 새로운 날짜가 만드는 게 아니다. 어제와 다른 생각 하나가 만든다."

## 제목 작성법

마케팅 기법 5종 적용하여 5개 제시:
1. 호기심 유발 2. 대립/반전 3. 직접 타겟팅 4. 질문형 5. 숫자/구체성
☐ 30자 이내 ☐ 호기심/긴장감 ☐ 클릭 유도 ☐ 본문 일치
